# Packtils
A Minecraft Modpack Utility.

### Features:
- Prompt the user to **Install Optifine** if not already installed
- Set A **Custom Title**
- Set A **Custom Icon**

To Set The Custom Icon Go To `{Instance Directory}\resources\assets\packtils\{icon_16.png or icon_32.png}`
